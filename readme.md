# Concesionaria AutoMax


Sitio web desarrollado con HTML5 y CSS puro para una concesionaria de autos.


## 📌 Características


- Estructura semántica HTML5
- Diseño responsive básico
- Catálogo de vehículos
- Formulario de contactoo
- Sección institucional


## 📁 Estructura


concesionaria/
│
├── index.html
├── css/
│ └── styles.css
└── img/
└── (imágenes de autos)


## 🚀 Uso


1. Descargar o clonar el proyecto
2. Abrir `index.html` en el navegador
3. Personalizar contenido e imágenes


## 🛠️ Tecnologías


- HTML5
- CSS3
